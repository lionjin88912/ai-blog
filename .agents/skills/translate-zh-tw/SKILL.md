# 翻譯與在地化 (Traditional Chinese - Taiwan)

## 概述

此 Skill 專門用於將文字精準翻譯為台灣繁體中文。不僅是語言轉換，更強調「在地化（Localization）」，確保術語、標點符號與語氣符合台灣使用者的習慣。

## 執行步驟 (Workflow)

1. **分析語境**：判斷輸入內容是屬於技術文件、程式代碼、商務溝通還是日常對話。
2. **查閱術語**：參考 [glossary.md](references/glossary.md) 確保關鍵詞彙（如 Optimization, Project, Software）翻譯正確。
3. **套用風格**：根據 [style-guide.md](references/style-guide.md) 調整語氣（您/你）與標點符號規範。
4. **檢查間距**：確保中英文、中數字之間有適當的半形空格。
5. **最終驗證**：檢查是否誤用了簡體中文慣用語（如：視頻、優化、質量、支持）。

## 使用範例

### 範例 1：翻譯技術文件段落
**User**: "This software provides an optimized way to manage your database clusters with high availability."
**Output**: 「這套軟體提供了一種最佳化方式，能以高可用性管理您的資料庫叢集。」

### 範例 2：翻譯 UI 選單
**User**: "Open folder in repository, settings, and advanced optimization."
**Output**: 「在儲存庫中開啟資料夾、設定、進階最佳化」
