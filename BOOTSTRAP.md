# BOOTSTRAP.md - 給行銷同仁的快速開始指南

_嗨,我是你的行銷內容助手。我可以幫你產出文章並自動發布到 WordPress,你不用懂任何技術。_

---

## 🏭 系統長怎樣(一張圖看懂)

```
你 ──說話──▶ 行銷內容工廠(我)── 派任務 ──▶ 通用寫手 ── 載入 ──▶ 人格資料
                                              │
                                              └─ 完成後發布到 ──▶ 該人格自己的 WordPress
```

- **行銷內容工廠**:你只需要跟它對話,它會處理所有事
- **通用寫手**:幕後執行寫作 SOP,你不用直接操作
- **人格資料**:目前內建「林太(58 歲退休教師,慢旅文化派)」,你可以自己生新人格
- **每個人格 = 一個品牌 = 一個 WordPress 部落格**:林太發 lifestyle blog、王老闆發美食 blog,各走各的,設定獨立。

## ✨ 我可以幫你做這 5 件事

| 模組 | 你會用到的時機 | 怎麼觸發 |
|---|---|---|
| 1️⃣ 設定某人格的 WordPress | 拿到新電腦/換 blog/重設密碼時 | 「設定林太的 WordPress」 |

## 🚀 三步開始

### 第一步:確認環境(你的電腦剛裝完才需要)

在終端機打:
```
check env
```

跑完後再打:
```
memory 記一下 env
```

### 第二步:設定林太的 WordPress(每個人格做一次,新裝預設先設林太)

直接打:
```
設定林太的 WordPress
```

我會逐項問你網址、帳號、應用程式密碼,3 分鐘搞定。
完成後我會自動發一篇測試草稿,確認連線通。

> 之後新增別的寫手人格(王老闆、小美…),會在建立人格的對話裡一起設定他們各自的 WordPress 部落格,不用回來重跑這一步。

### 第三步:寫第一篇

```
用林太寫一篇九份兩天一夜的文章
```

我會幫你做完查資料、抓圖、寫作、SEO、發布草稿,2-3 分鐘看到結果。
草稿會出現在 WordPress 後台,你檢查潤稿後按「發布」就上線。

## 💡 想新增一位寫手?

不用找工程師,你自己就能生。對我說:

```
我想新增一個寫美食的人格
```

我會問你 8 題(這位寫手的名字、年紀、口吻、風格…),3-5 分鐘後新人格就建立好了,可以立刻用:

```
用王老闆寫一篇饒河夜市
```

每個新人格都會用通用寫手的 SOP(查資料/抓圖/SEO/發布),只是口吻、選材會依照你定義的個性自動套用。

---

## 🆘 出問題怎麼辦?

直接告訴我「失敗了」「跑不出來」「報錯了」,我會帶你看訊息、對照排錯表處理。常見問題:

- 帳密不對 → 重新設定 WordPress
- 連不上網站 → 檢查公司網路有沒有擋
- 圖片抓不到 → 重試一次,還是不行就先存草稿
- 文章發出去版面亂 → 把連結貼來,我重新產生一次
