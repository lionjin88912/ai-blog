# BOOTSTRAP.md - 給行銷同仁的快速開始指南

_嗨,我是你的行銷內容助手。我可以幫你產出文章並自動發布到 WordPress,你不用懂任何技術。_

---

## 🏭 系統長怎樣(一張圖看懂)

```
你 ──說話──▶ 行銷內容工廠(我)── 派任務 ──▶ 通用寫手 ── 載入 ──▶ 人格資料
                                              │
                                              └─ 完成後發布到 ──▶ 該人格自己的 WordPress
```

- **行銷內容工廠**:你只需要跟它對話,它會處理所有事
- **通用寫手**:幕後執行寫作 SOP,你不用直接操作
- **人格資料**:目前內建「林太(58 歲退休教師,慢旅文化派)」,你可以自己生新人格
- **每個人格 = 一個品牌 = 一個 WordPress 部落格**:林太發 lifestyle blog、王老闆發美食 blog,各走各的,設定獨立。

## 🚀 兩步開始(其實只要打一句話)

### 第一步:打開瀏覽器終端機,啟動 Gemini

雙擊 `ai-sandbox-darwin-arm64`(或 Windows 上的 .exe),瀏覽器自動跳出後在裡面打:

```
gemini
```

第一次會問你登入,選 `Sign in with Google` 用公司 Gmail 登入。

### 第二步:**隨便打一句話**

直接打,例如:

```
如何開始
```

或更直接:

```
我要寫一篇行銷文章
```

接下來我(Gemini)會自動處理所有事情,你不用記任何指令:

```
1. 第一次的話,我會幫你準備環境(30 秒)
2. 列出我能做什麼
3. 顯示 5 項選單,你回數字就好
```

從這裡開始就跟著我的引導走 — 設定 WordPress、寫文章、新增人格…全程對話式,不會再要求你打任何技術指令。

> 💡 **如果是同事 IT 人員幫忙裝的**,他可能會打 `check env`、`/skills list` 這些指令做檢查 — 那是工程驗證用的,你日常用不到。

## 💡 想新增一位寫手?

不用找工程師,你自己就能生。對我說:

```
我想新增一個寫美食的人格
```

我會問你 8 題(這位寫手的名字、年紀、口吻、風格…),3-5 分鐘後新人格就建立好了,可以立刻用:

```
用王老闆寫一篇饒河夜市
```

每個新人格都會用通用寫手的 SOP(查資料/抓圖/SEO/發布),只是口吻、選材會依照你定義的個性自動套用。

---

## 🆘 出問題怎麼辦?

直接告訴我「失敗了」「跑不出來」「報錯了」,我會帶你看訊息、對照排錯表處理。常見問題:

- 帳密不對 → 重新設定 WordPress
- 連不上網站 → 檢查公司網路有沒有擋
- 圖片抓不到 → 重試一次,還是不行就先存草稿
- 文章發出去版面亂 → 把連結貼來,我重新產生一次
